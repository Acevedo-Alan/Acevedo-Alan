##  Proyectos Destacados:

### RDAM — Registro de Deudores Alimenticios Morosos
> **Cliente:** Poder Judicial de Santa Fe · **Programa:** Campus 2026

Plataforma seleccionada entre los **8 finalistas** del programa Campus 2026 para la fase de evaluación final.

- Diseño e implementación de una **API RESTful** con Java y Spring Boot bajo arquitectura de capas (Controller → Service → Repository).
- Modelado y gestión de base de datos relacional con **PostgreSQL**, garantizando integridad de los registros.
- Pruebas de integración de endpoints con **Postman** para validar el flujo de datos entre cliente y servidor.

### OPENLODGE
> **Proyecto académico colaborativo**

Aplicación full-stack desarrollada en equipo con flujo de trabajo profesional en Git.

- Construcción del frontend con **HTML, CSS, JavaScript y TypeScript**, conectado a la API mediante `fetch`/`axios`.
